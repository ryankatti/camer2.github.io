PermissionService provides the functionality to check and request the Permissions required for the BlueJeans SDK. This service is an optional service, you can choose to use your own permission mechanism.

BlueJeans SDK requires two permissions Manifest.permission.RECORD_AUDIO and Manifest.permission.CAMERA to join a call.

The app needs to have mandatory permission android.Manifest.permission.RECORD_AUDIO in application manifest out of two permissions that need to be requested.

Note : Registering the service in onCreate of an activity is mandatory before calling request permissions within that activity.
